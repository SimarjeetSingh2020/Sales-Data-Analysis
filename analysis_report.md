# Sales Data Analysis Report

## Objective
Analyze sales data to calculate key metrics and identify the best-selling product.

## Steps Performed
1. Loaded CSV data using pandas
2. Inspected dataset structure and data types
3. Cleaned missing and invalid values
4. Calculated total revenue, average sales, and highest sale
5. Identified best-selling product

## Key Findings
- Total Revenue was calculated using sum()
- Average sales provided insight into typical transaction value
- Best-selling product was identified using groupby()

## Conclusion
This analysis demonstrates basic real-world data analysis using pandas, including data cleaning, aggregation, and reporting.
